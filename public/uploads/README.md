Place site-served local media here and reference it from Excel with paths like `/uploads/...`.

Recommended structure:

- `public/uploads/alcantara/panel/`
- `public/uploads/alcantara/cover/`
- `public/uploads/alcantara/swatches/`
- `public/uploads/alcantara/cases/`

Do not put absolute filesystem paths such as `/Users/...` into the workbook.
